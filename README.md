# Binus TechFest 2024 - Data Analytics Competition

## Overview
Proyek ini dibuat untuk mengikuti kompetisi Binus Techfest 2024 pada bidang Data Analytics.
Tujuan proyek ini adalah melakukan analisis data sosial-ekonomi tingkat provinsi serta menyiapkan dataset yang bersih dan siap digunakan untuk tahap modeling.

## Dataset
Dataset berisi data sosial dan ekonomi tingkat provinsi di Indonesia, meliputi:
- PDRB dan PDRB per Kapita
- Indeks Pembangunan Manusia (IPM)
- Jumlah penduduk
Anggaran APBN dan TKDD
Persentase penduduk miskin

## Methodology
Tahapan analisis yang dilakukan:
1. Exploratory Data Analysis (EDA)
2. Pemerikasaan korelasi dan anomali data
3. Penanganan missing value (Interpolaso, Rasio, KNN Imputation)
